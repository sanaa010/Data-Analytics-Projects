# 🚚 Supply Chain Delivery Performance Analysis

## 📌 Overview
This project focuses on analyzing the delivery performance of a global e-commerce company using **Python, Data Analytics, and Machine Learning**.

The objective was to identify the major causes of delayed deliveries, analyze operational bottlenecks, and build a predictive model to detect high-risk orders before shipment.

The project includes:
- 📂 Data Loading
- 🧹 Data Cleaning
- 📊 Exploratory Data Analysis (EDA)
- 📈 Data Visualization
- 🤖 Machine Learning
- 💡 Business Recommendations

---

# 🗂️ Dataset Information

- **Dataset:** DataCo Supply Chain Dataset
- **Total Records:** 172,765 Orders
- **Time Period:** 2015 – 2018

### Dataset Contains:
- Order Details
- Shipping Information
- Delivery Status
- Customer Segments
- Product Categories
- Regional Data
- Profitability Metrics

# 📌 Key Insights

## 🚨 Major Findings
- ⚠️ **54.71%** of orders were delivered late
- 🚚 First Class shipping showed the highest delay rate
- 📅 Seasonal demand spikes increased delays
- 💳 Payment review issues affected delivery performance
